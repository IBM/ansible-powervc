Plugins
=======

The **IBM PowerVC collection** contains plugins that can be used in a playbook
to automate tasks over PowerVC. Ansible executes each tasks on the target node and
returns the result back to the controller. 

Plugin reference
----------------

Reference material for each plugin contains documentation on the accepted
parameters and their expected values.
